Nessa aula iremos estudar os Operadores:

---------------------------------------------------------------------

Aritmeticos

Os operadores aritmeticos são: +, -, *, /, %, **

---------------------------------------------------------------------
Atribuidores

var n = 4 | n = n + 4 | n += 4 | n++ ou n--

------------------------------------------------------------------
Relacionais

Os operadores relacionais são: > | < | >= | <= | == | !=

Para toda expressão que tenha um valor relacional ligado a ela, o resultado dessa expressão será um boolean.

Os dois iguais (==) se lê: iguais ou igual. Ex: 5 == 5 (cinco é igual a cinco)

O sinal de exclamação e igual (!=) le sê: não igual ou diferente. Ex: (4 != 5) nesse caso 4 é diferente de 5.

(5 == 5) true

(5 == '5') true

(5 === '5') false, os três iguais significa se eles são identicos com o mesmo valor e o mesmo tipo.

---------------------------------------------------------------------
Lógicos

Os operadores lógicos são: ! | && | ||

A exclamação (!) quer dizer NEGAÇÃO

O não (!) é um operador unário, so tem um operando. Depois da exclamação ou eu coloco TRUE ou FALSO ou uma expressçao que dê verdadeiro ou falso.

Dois && comerciais quer dizer uma CONJUNÇÃO, nosso e lógico.

Um conjunção é um operador BINÁRIO, eu tenho dois valores logicos, um de cada lado. Na conjunção basta que um dos lados seja falso pro resultado ser Falso.

As duas barras (||) quer dizer uma DISJUNÇÃO, nosso ou lógico.

A disjunção também é um operador BINÁRIO, que tem dois valores lógicos, um de cada lado.Na disjunção basta que um dos lados seja verdadeiro pro resultado ser verdadeiro.
