Primeiros passos com DOM

O que é o DOM?

Basicamente é um ácrono para Document Object Model

É um modelo de objetos para documentos, um conjunto de objetos dentro do meu navegador que vai dar acesso aos componentes internos do meu site.

O que é Árvore DOM?

Árvore DOM começa da raiz, essa raiz no navegador chamamos de window. Dentro do window eu tenho vários outros objetos como: localion, documents e history

Dentro de Documents eu tenho o HTML e dentro de HTML eu tenho o head e body.

Dentro de head eu tenho inúmeras tags como title meta etc.

Já dentro do body eu tbm posso ter inúmeras tags como h2, h3, p, div.

Dentro de p eu tenho um filho chamado strong.

Essa é a árvore hierarquica do site que acabei de criar.

obs: Eu posso navegar dentro dessa árvore, existindo várias maneiras de navegar dentro de um elemento.

Selecionando os Elementos

Existe inúmeras metodos para acessar esses elementos como por exemplo por Marca, por Id, por Nome, por Classe, por Seletor.
